# 文件上传与存储（零代码）

## 参数声明
```yaml
request:
  file:
    files: { required: true, maxSize: 32MB, types: ["image/png","image/jpeg"] }
```

## 解析兼容
- `multipart/form-data`：自动解析表单与文件
- 原始体上传：当无法解析为多部件时，框架将请求体视为单文件 `infile`

## 保存到 Blob
```yaml
- upload.save: { ds: local, input: "{{ file.files }}", naming: sha256, out: ids }
```
- `naming`: `sha256|uuid|original`
- 目录：`configs/project.yaml` 中 `blob.local.basePath`（默认 `storage/`）
