# 测试与校验（零代码）

## Lint
- 使用步骤 `admin.lint` 获取接口配置检查结果，快速发现缺失字段或不一致。

## 集成测试
- 通过 PowerShell/脚本文件发送请求；示例脚本位于 `test_*.ps1` 与 `scripts/`。
- 覆盖场景：JWT/ApiKey、限流、事务、上传、插件调用、向量检索。

## 调试技巧
- 打印 `ctx.vars.sql_log` 观察 SQL 执行轨迹与参数绑定顺序。
- 在 `response.headers` 中加上 `X-Trace` 等标识字段辅助定位。

## 热重载验证
- 修改 `configs/interfaces` 后观察日志中重载摘要（新增/移除/未变）。
